## ✅ 포함된 과제 / 학습 내용  
- 개념적 설계 (ERD)  
- 논리적 설계 및 정규화  
- SQL 스크립트 작성 및 실행  
- 테이블 명세서 작성  
- 과제 제출을 위한 압축 파일 정리  

## 🛠️ 사용 / 실행 방법  
1. 로컬 또는 학교 DB 환경(MySQL, MariaDB, 혹은 다른 RDBMS)에 접속합니다.  
2. 원하는 `.sql` 파일을 실행하여 테이블 생성 및 데이터 삽입을 수행하세요.  
3. 필요에 따라 `.drawio` 파일로 ERD를 열어 구조를 확인할 수 있습니다.  
4. `.pptx` 또는 `.pdf` 파일을 열어 설계 문서 및 정규화 과정을 참고할 수 있습니다.  

## 🎯 목적과 의도  
- 데이터베이스 설계 및 구축 과정을 체계적으로 정리하고 복습하기 위함  
- ERD → 논리/물리 설계 → SQL 구현까지 일련의 흐름을 경험하고 숙달하기 위함  
- 추후 다른 프로젝트(예: Java 기반 키오스크, 웹 DB 연동 등)에서 참고 가능한 설계 템플릿이 되도록  

## ✨ 앞으로의 계획 / 확장 아이디어  
- 기존 설계/SQL에 CRUD, JOIN, 트랜잭션 예제 추가  
- ERD를 기반으로 한 ORM 모델 설계 (예: Java JPA, Hibernate)  
- 간단한 데이터 시각화 또는 API 연동 실습 프로젝트로 확장
